# Getting Started

## Option 1: Browser
  Visit [Atlas](https://atlas.nomic.ai/) to add your dataset via your internet browser.

## Option 2: Python

Call `map_data` from the Atlas Python SDK to add your dataset programmatically.

Your dataset result will contain a link to your map in your [Atlas Dashboard](https://atlas.nomic.ai/data).

  ```bash
  pip install nomic
  ```

  ```python
  from nomic.atlas import map_data

  dataset_result = map_data(
    data=data,
    ...
  )
  ```
