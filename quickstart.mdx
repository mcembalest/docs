Storing data in Atlas:  
<AccordionGroup>
  <Accordion title="Drag & Drop" icon="chrome">
    Add data to Atlas from browser

  </Accordion>

  <Accordion title="Python SDK" icon="python">
    1. Install the Python client:
    ```bash
    pip install nomic
    ```

    2. Call the `map_data` function with your data to embed & any associated metadata:

    ```python
    from nomic.atlas import map_data

    dataset_result = map_data(
      data=data,
      ...
    )
    ```
  </Accordion>
</AccordionGroup>
