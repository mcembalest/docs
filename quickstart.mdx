# Getting Started

## Option 1: Drag+drop data in browser
  Add data to Atlas from browser
## Option 2: Python

Call `map_data` from the Nomic Atlas python SDK to upload data to Atlas

  ```bash
  pip install nomic
  ```

  ```python
  from nomic.atlas import map_data

  dataset_result = map_data(
    data=data,
    ...
  )
  ```
